Enable https support for thatyear.me

1.证书选择

https 证书有很多颁发者，处于费用的考虑（毕竟这个博客已经从日更变为月更到现在是年更），没必要每年花费个几百上千的去买证书，于是就是用了 Let’s Encrypt 提供的免费 https 证书。

证书的生成在Let’s Encrypt 网站上有详细的步骤说明。对于绝大多数的主机，只需要在 cert 上选择操作系统以及 http 服务器的类型，就可以通过脚本一键生成证书了。

https://certbot.eff.org/#centosrhel7-nginx

2.nginx编译

编译 nginx 的时候需要参数 –with-http_ssl_module
参考官网配置

3.问题

执行过程中可能需要依赖库
证书有效期是90天，需要自动更新
